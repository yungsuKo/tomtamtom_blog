애니메이션은 요소에 적용되는 css스타일을 다른 css스타일로 부드럽게 전환시켜줄 때 사용한다.
css 애니메이션은 기존에 사용되던 스크립트를 이용한 애니메이션보다 아래 3가지 이유에서 이점을 가져다준다.

1. Javascript를 모들더라도 애니메이션을 만들 수 있음.
2. frame-skipping 같은 여러 기술을 이용하여 부드럽게 렌더링됨.
3. 브라우저에서의 애니메이션 성능을 효율적으로 최적화할 수 있음.
